Опис курсу

На цьому курсі ви навчитеся створювати веб-сайти з використанням HTML та CSS. Ви дізнаєтеся, як створювати веб-сторінки, розташовувати елементи на сторінці, додавати зображення, стилізувати текст та створювати анімацію. Після закінчення курсу ви зможете створювати власні веб-сайти та розміщувати їх в Інтернеті.

Список тем

  1. Введення до HTML та CSS
  2. Робота з тегами та елементами HTML
  3. Робота з CSS та стилізація тексту
  4. Розташування елементів на сторінці
  5. Додавання зображень та створення анімації
